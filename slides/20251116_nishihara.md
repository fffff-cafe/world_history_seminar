<a id="approve"></a>
## なぜ米国が「承認」するか（要因）
- 原潜技術・燃料は拡散リスクが高く、同盟内で管理・統制する（例: Devdiscourse: https://www.devdiscourse.com/article/law-order/3633748-us-allows-south-korea-to-build-nuclear-powered-submarine）
- 戦略上の信頼・相互運用性・影響力を維持する（上記報道各紙）
- 地域抑止力を同盟国で底上げする（対中・対北を意識）

<a id="holders"></a>
## 原潜を保有・運用する主な国（概観）
- 米・露・英・仏・中・印が原潜を保有する。役割はSSBN（抑止）とSSN/SSGN（攻撃・特作）に大別される（各クラスのWikipediaに準拠）
- 統計では、2024年時点でアメリカが約66隻、ロシアが約31隻、英国10隻、フランス9隻、中国12隻、インド2隻などという試算（公開推計の一例）


---

<a id="holders-why-few"></a>
### なぜ保有国が少ないのか

- 技術的・インフラ的ハードル（原子炉設計・安全管理・船体構造・静粛性確保・整備施設等）
- 燃料・核材料管理・高度な人材・潜水艦運用ノウハウ
- 高コスト：建造・運用・維持に莫大な費用がかかる
- 地政学的・国際的な監視・非拡散義務等の制約

---

<a id="holders-self-reliant"></a>
### そもそも原潜が各国独力で作れるのか

- 海軍用原子炉：小型高出力・静粛・長寿命炉心の設計と実証が難
- 燃料サイクル：濃縮・製造・管理・廃棄まで一式を自前整備
- 安全・規制・インフラ：安全文化、規制枠組み、事故対応、専用ドック等
- 造船・静粛化：圧力殻、遮蔽、低騒音推進、振動隔離
- 統合・人材・維持費：システム統合、要員養成、燃料交換・廃炉まで巨費
- 国際制約：輸出管理・拡散防止・外交コスト

結論：完全独力は例外的で、実際は同盟・技術移転に依存しがち。

---

<a id="holders-abbrev"></a>
### 主な分類・略号例

 - **SSN**：Submarine, Ship Nuclear（あるいはサブマリン、原子力推進、攻撃型）＝原子力推進の攻撃型潜水艦。 （米海軍: https://www.history.navy.mil/research/library/online-reading-room/title-list-alphabetically/c/current-doctrine-submarines.html）
 - **SSBN**：Submarine, Ship Ballistic Nuclear（原子力推進、弾道ミサイル搭載型）＝戦略核ミサイルを搭載可能な原潜。 （CSP: https://www.csp.navy.mil/）
 - **SSGN**：Submarine, Ship Guided Nuclear（原子力推進、巡航ミサイル搭載型）等、特定用途型もあります。

<a id="basics"></a>
## 原潜の基本
- 定義：原子炉推進で長時間潜航する艦を指す（Wikipedia: https://en.wikipedia.org/wiki/Nuclear-powered_submarine）
- 種別：SSBN=核抑止の中核／SSN=攻撃・偵察・特作／SSGN=巡航ミサイル主体
- 強み：長い行動持続、静粛性、戦略的不可視性
- 制約：食料・整備・弾薬・人員・法的枠組み・コスト

## 通信（本国→原潜）
- ELF/VLFは深深度で一方向・低帯域通知に用いる（Wikipedia: https://en.wikipedia.org/wiki/Communication_with_submarines）
- ブイ・浮上・衛星は双方向・高帯域だが露出が増える（同上）
- 実運用は受信中心で必要時のみ露出する

<a id="endurance"></a>
## 「無限航行」の誤解と実際の制約
- 食料・消耗品・弾薬が先に尽きる。補給・休養が必要である
- 定期整備・故障対処・船殻/機器の摩耗が制約となる
- 原子炉燃料は長寿命でも無限ではない（Virginia級は長寿命炉心設計とされる：Wikipedia: https://en.wikipedia.org/wiki/Virginia-class_submarine）
- 乗員の心理・医療上の限界が行動を制約する（History Collection事例: https://historycollection.com/this-60-day-submarine-voyage-in-1960-tested-the-psychological-and-physical-limits-of-crew/）

<a id="detection"></a>
## 探知：音響と非音響
- 音響は艦/ヘリ/固定線のソナー、静粛化競争が続く
- 非音響は衛星・光学・SAR・磁気・環境の総合活用（CSIS: https://ontheradar.csis.org/issue-briefs/non-acoustic-submarine-detection/）
- 研究・事例は増加中（Naval Sub League: https://archive.navalsubleague.org/1990/nonacoustic-means-of-submarine-detection）
- 将来はリモート海中監視網とAI融合で検出確率が環境依存で変動する（DSTG概説: https://www.dst.defence.gov.au/sites/default/files/events/documents/Insights%20Paper%20-%20Remote%20Undersea%20Surveillance%20F1.pdf）

<a id="quiz"></a>
## クイズ
- Q1. 領海を外国の一般船舶が通過することはOKか？
- Q2. 領海を外国の潜水艦が通過することはOKか？
- Q3. EEZを外国の一般船舶が通過することはOKか？
- Q4. EEZを外国の潜水艦が通過することはOKか？

---

<a id="answers"></a>
## 解答
- A1（Q1の解答）: 原則OKである。無害通航権が適用される（脅威・演習・情報収集など無害性を損なう行為は不可）。
- A2（Q2の解答）: 条件付きでOKである。潜水艦は領海では浮上し国旗を掲げて通航する義務がある。潜航したままの通過は不可。
- A3（Q3の解答）: OKである。EEZでは航行の自由が認められる。
- A4（Q4の解答）: OKである。EEZでは潜水艦の潜航通過も一般に許容される。ただし特定の活動は緊張や摩擦を生むため注意が必要である。
  - EEZは広いので、領海ギリギリを通過するなどは緊張が高まる
  - 「天然資源の探査・開発・保存・管理」に関する主権的権利（56条）があるので、資源探査や海底調査はNG

<a id="law"></a>
## 法・慣習：やってよい/悪い
- 公海での活動、共同演習、条件次第でEEZ通過は許容される場合がある（一般論）
- 他国領海の無断潜航・偵察や拡散助長は違法/グレーである（一般論）

### 実務で起きがちなグレー
- 近隣国沿岸への秘匿接近・索敵、EEZ境界近傍での待機・監視
- 静粛試験やデータ収集のための環境利用（公開されにくい）

<a id="japan"></a>
## 日本が原潜を持つ論点（現状：未保有）
- 法・政策：憲法/非核三原則/運用指針との整合を要する
- 技術・体制：原子炉・燃料・整備・人材・安全文化の整備が前提
- 戦略：地域バランス・同盟調整・費用対効果で評価する

<a id="summary"></a>
## まとめ（3行）
- 原潜は長持続×静粛で戦略価値が高いが無限ではない
- 拡散・法・運用・コストが実装のボトルネックである
- 米国は同盟・産業・抑止の観点から厳格に関与する
